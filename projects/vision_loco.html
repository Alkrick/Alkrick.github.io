Vision-Based Quadrupedal Locomotion

Mohamed Al-Khulaqui

Intro

An RL-Framework is developed to enable legged robots to navigate unstructured terrain without the need for external perception. In order to compensate for the lack of external perception an Implict-Explict Estimator module is proposed; The IE Estimator provides explict body velocity estimation for the robot as well as implict state encoding of the robot and terrain. The policy is trained using PPO.
